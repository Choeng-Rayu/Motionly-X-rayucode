# Quickstart

## Run Motionly

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.

## Edit A Project

1. Select a layer from the scene panel or click an object on the canvas.
2. Drag the object or adjust position, scale, rotation, and opacity in Properties.
3. Change text, size, color, duration, delay, and easing.
4. Press Play or scrub the timeline to preview frame by frame.
5. Save the project back to `.motion`.

Motionly serializes visual edits into the `.motion` source automatically.
