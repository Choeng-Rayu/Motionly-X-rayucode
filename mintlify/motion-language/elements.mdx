# Elements

Motionly elements are named objects that can be selected, positioned, animated, and serialized.

Common properties:

- `x`
- `y`
- `scale`
- `rotation`
- `opacity`
- `blur`
- `width`
- `color`
- `center`

## Image Or SVG Asset

```motion
import "/video-motion/assets/motionly/logo.svg" as motionlyLogo

motionlyLogo {
  center
  width 220
  opacity 1
}
```

Use clear names so layers are readable in the editor and in source review.
