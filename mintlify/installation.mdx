# Installation

## Requirements

- Node.js `20.19.0` or newer
- npm
- A modern Chromium-based browser for the best export support

## Install

```bash
git clone https://github.com/PromSereyreaksa/motionly.git
cd motionly
npm install
```

## Start The Editor

```bash
npm run dev
```

Then open the local URL printed by Vite.
