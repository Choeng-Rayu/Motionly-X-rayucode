# Animation Presets

Motionly favors a small set of subtle presets:

- Fade in
- Rise in
- Scale in
- Blur reveal
- Soft drift

Default to `power3.out` for smooth professional motion.

Presets should make a subject easier to read. Avoid stacking multiple strong effects on the same object unless the scene needs a deliberate transition.
