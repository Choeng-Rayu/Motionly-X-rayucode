# Text Presets

Text animation should preserve readability.

Good defaults:

- Fade title from `opacity 0`
- Rise title from `y 80`
- Reveal text with light `blur 12`
- Stagger subtitle after the title

Use short durations for interface labels and longer durations for hero copy.
