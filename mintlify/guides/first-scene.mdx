# Build Your First Scene

Start with one focal subject, one supporting text layer, and one simple entrance.

```motion
canvas {
  size 1920x1080
  fps 60
  duration 5s
  background #020308
}

text title {
  value "Motionly"
  center
  size 96
  color #ffffff
}
```

Add a rise-in animation and preview the result in the editor.
