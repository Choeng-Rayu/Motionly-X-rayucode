# Motionly

Motionly is a browser-first motion graphics editor built around readable `.motion` files. Create scenes visually, tune animation timing, preview the result, and export motion graphics without writing application code.

## What Motionly Does

- Edits `.motion` projects through a visual canvas and timeline
- Keeps animation source readable and portable
- Supports text, images, SVG assets, scene timing, and animation presets
- Exports browser-rendered motion graphics

## Core Idea

`.motion` is the saved source format. The editor is the normal way to create and adjust projects, while the file format stays simple enough for review, examples, and agent-assisted authoring.
