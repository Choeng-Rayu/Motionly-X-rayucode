# Motionly UI Guide

Motionly uses one focused editor screen.

## Top Toolbar

- Open a `.motion` project
- Save the current project
- Load preset animations with confirmation dialog
- Export MP4 with progress feedback
- Open and star the GitHub repository

## Left Navigation Rail

Icon-based tabs for quick access to tools and content:

- **Media** - Assets panel with organized folders (Images & Graphics)
- **Audio** - Audio assets panel
- **Text** - Text tools
- **Effects** - Animation presets library (text, object, transition, camera)
- **Scenes** - Scene backgrounds and overlays
- **Adjustments** - Color and timing adjustments
- **Settings** - Project settings

## Assets Panel

Browse and manage project assets:

- Click thumbnails for full-screen preview (temporary, no timeline disruption)
- Organized folder structure showing audio and visual assets
- Drag assets directly onto timeline to create clips
- Green drop indicator shows where clips will be placed

## Presets

Browse animation templates:

- Animated GIF preview thumbnails
- Load preset with confirmation dialog (red warning button)
- Presets load as fully editable projects

## Preview

The center panel renders the scene at the project's aspect ratio.

- Click an object to select it
- Drag a selected object to move it
- Drag corner handles to scale it
- Click empty space to deselect
- Use Fit, zoom, and fullscreen to inspect the canvas

## Properties

The right panel edits the selected layer without requiring source changes.

- Custom polished sliders for smooth value control
- Styled inputs for precise values
- Visual preset cards for animation selection
- Position, scale, width, rotation, and opacity
- Text content, font size, and color
- Scene background color
- Animation preset, duration, delay, and easing

## Timeline

The lower panel contains playback and role-based clip editing.

- Persistent overlay tracks appear above the gap-free Main Track; audio tracks appear below it.
- Compatible non-overlapping clips reuse a purpose track, while real overlaps create another overlay.
- Scrubbable ruler and frame-accurate playhead anchor Split and trim operations.
- Drag clips horizontally to reorder/time them and vertically between compatible tracks.
- Trim handles work for media, text, images, overlays, and effects.
- Magnet toggles clip-edge/playhead snapping and Main Track ripple behavior.
- Undo/Redo buttons and standard keyboard shortcuts restore serialized `.motion` snapshots.
- Timeline `−`/`+` controls and `Ctrl/Cmd` + wheel/pinch zoom around the pointer.
- Eye and speaker controls persist track visibility and mute state.
- Dragging Assets creates thumbnail clips and writes track/timing metadata automatically.
- The panel divider remains vertically resizable.

## `.motion` Source

The `.motion` button opens the generated source for inspection. Visual editing remains the primary workflow.
