# Installation

## Requirements

- Node.js `20.19.0` or newer
- npm
- A modern Chromium-based browser for best MP4 export support

## Install

```bash
git clone https://github.com/COPPSARY/Motionly.git
cd Motionly
npm install
```

## Run

```bash
npm run dev
```

Open the local Vite URL in your browser.

## Build

```bash
npm run build
```

## Test

```bash
npm run test:run
```
